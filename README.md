# Piper Robot Control Project

## Introduction

This project is code for controlling a Piper robot.

It allows users to connect to the robot, control its movements, operate its joints, and test robot commands through a Python-based program.

## Main Features

* Connect and initialize the Piper robot
* Control robot joints and movements
* Execute user-defined robot commands
* Check robot status during operation
* Test basic robot control functions

## How to Use

1. Download

```bash
git clone https://github.com/KGW218/robotics-project.git
cd robotics-project
```

2. Run

```bash
python main.py
```

If required packages are included, install them first:

```bash
pip install -r requirements.txt
```
